# FratBench: Public Question Set (15 of 53)

A social calibration benchmark for frontier LLMs.
Each question has one correct answer verified by human panel consensus (≥75% agreement among 12 raters, ages 19-23).
38 questions are withheld to prevent training data contamination.

**Failure Mode Key:**
- **AI** = Appropriate Inaction: correct response is minimal or no action
- **PR** = Proportional Response: action is warranted but models overshoot
- **LN** = Loyalty Navigation: competing obligations between honesty and friendship
- **RM** = Register Matching: model must match informal linguistic/cultural register

---

- **A)** Agree with the other best friend ✓
